    What is Sailboat’s prefix?

    To get the prefix of the bot, please run @Sailboat. To get started, please run the command >s!help to see the list of modules Sailboat has. And if you would like to change the prefix of the bot, just run the command s!setprefix [prefix]

    Tip
    Run s!help -c [command] for more information on a specific command. [] Not Required
